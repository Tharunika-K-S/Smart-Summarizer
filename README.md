\# Smart Summarizer



A Flask-based AI application that summarizes:

\- PDFs

\- URLs

\- Images (OCR)



\## Features

\- Text summarization (short, detailed, bullet)

\- History tracking

\- Audio output

\- PDF download



\## Tech Stack

\- Flask

\- Transformers (HuggingFace)

\- SQLite

\- OCR (Tesseract)



\## How to Run

pip install -r requirements.txt

python app.py
